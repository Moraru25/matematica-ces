🎮 Matematica Magică - Joc pentru Clasa a V-a (CES)

Am creat un joc educațional complet în Python (tkinter) special adaptat pentru elevii cu Cerințe Educaționale Speciale (CES).

✅ Caracteristici principale:

1. Interfață accesibilă:
  • Fonturi mari (16-36px) pentru citire ușoară
  • Culori cu contrast ridicat (verde/bleu/galben)
  • Butoane mari și etichetate clar
  • Layout simplu și intuitiv
2. Reprezentări vizuale pentru toate operațiile:
  • Adunare: Grupuri de cercuri colorate care se combină
  • Scădere: Elemente vizuale cu "tăiere" (crossed out)
  • Înmulțire: Grupuri repetitive (ex: 3×4 = 3 grupuri cu câte 4)
  • Împărțire: Distribuirea în grupuri egale
3. Sistem de progresie:
  • 5 nivele de dificultate crescătoare (1-10, apoi 1-20, 1-50, 1-100)
  • Trecerea la nivelul următor după 5 răspunsuri corecte consecutive
  • Scor și sistem de recompense
4. Suport pentru învățare:
  • 3 indicii disponibile per problemă:
    • Indiciu 1: Interval de numere
    • Indiciu 2: Par/impar și ultima cifră
    • Indiciu 3: Valoare aproximativă
  • Feedback pozitiv și încurajator (fără penalizări severe pentru greșeli)
  • Mesaje motivaționale rotative
5. Adaptări CES:
  • Casetă de text mare pentru introducere ușoară
  • Posibilitatea de a alege tipul de problemă (adunare/scădere/înmulțire/împărțire/aleatoriu)
  • Progres vizual (bară de progres)
  • Ecran de bun venit cu instrucțiuni clare

Văd problema! WordPress.com (bibliotecaspecial.wordpress.com) nu permite încărcarea fișierelor HTML direct pe site. Acesta este un serviciu găzduit (hosted) care are restricții de securitate.

🔧 Soluții pentru WordPress.com:

Opțiunea 1 - GitHub Pages (GRATUIT și Recomandat)

Cel mai simplu pentru tine:
  1. Crează cont pe GitHub (github.com) - gratuit
  2. Crează un repository nou (ex: matematica-ces)
  3. Încarcă fișierul HTML acolo
  4. Activează GitHub Pages în setări
  5. Primești un link gen: https://numele-tau.github.io/matematica-ces/
  6. Pune acest link pe site-ul WordPress al bibliotecii

Opțiunea 2 - Netlify (GRATUIT)

  1. Netlify.com - drag & drop fișierul HTML
  2. Primești link instant
  3. Poți personaliza domeniul

Opțiunea 3 - Google Drive ( temporar)

  1. Încarcă fișierul pe Google Drive
  2. Setează ca "Oricine cu linkul"
  3. Folosește un convertor pentru a face linkul direct

🚀 Cel mai rapid: GitHub Pages
